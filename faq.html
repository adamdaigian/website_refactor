Frequently asked questions

What is an agent?

Agent is a small executable that will be installed on your server. The agent will monitor your server health.

What does the agent do?

The agent collects performance details from your server such as cpu usage, memory consumption, disk writes etc. Agent sends these data to Appedo servers every minute.

How can I view the server health?

Appedo provides unified dashboard. Using our dashboard you can view the server health from anywhere any time.

What is deep dive diagnostics?

Appedo can pin point the bottlenecks in your Java/.Net applications to the package, class and function level. You can view the slow queries as well. This will provide a complete stack trace of your application call on the server side.

What is load testing?

Appedo can generate synthetic user requests from cloud to hit your server. This way you can test whether your application can withstand 1000s of concurrent users.

What is end user monitoring?

Appedo provides nodes across the globe. You can get the performance of your web site or web application from many cities across the globe. We provide real browser view of your app from these global locations.

Can I try Appedo? Is there an eval version?

Appedo provides a free lite version. You can use that forever. This has the same feature set as full Appedo. Lite version can store only last 1 hour data.

Where are Appedo servers?

Appedo servers are on Amazon cloud.

Can I use Appedo in my secure premise?

Appedo provides cloud version as well as on-prem version.
